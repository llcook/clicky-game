# Clicky Game

This simple React game prompts the user to click an image block for a point. Repeat image clicks reset the score to zero.

Check it out here on [Github Pages](https://llcook.github.io/clicky-game/).

### Technologies:
* React
* Bootstrap CDN
* Github Pages
